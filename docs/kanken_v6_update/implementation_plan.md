# Kanken 分配シミュレーション 参照ファイル V6 更新およびエラーハンドリングの強化

本計画では、データ集計プログラム `backend_calc.py` にて参照するファイルを最新の `Kanken23510_2026年版_v6.xlsx` へ変更（現在すでに設定済みであることを確認）するとともに、ファイル読み込み時のエラーハンドリングを強化し、非エンジニアのユーザー様でもトラブルの原因がひと目で分かるように改善します。

## ユーザー確認事項

> [!NOTE]
> プログラム内の参照先ファイルパスは、すでに最新 of `Kanken23510_2026年版_v6.xlsx` を指すように設定されていることを確認いたしました。
> 今回は、処理が安全かつわかりやすく実行できるよう、エラー発生時の日本語メッセージ表示機能を追加した改善版コードを納品いたします。

## 提案する変更内容

### プログラムの堅牢化 (Error Handling)

#### [MODIFY] [backend_calc.py](file:///c:/Users/kxnxg/OneDrive/デスクトップ/シミュ２/backend_calc.py)
- Excelファイル（`Kanken23510_2026年版_v6.xlsx`、`分配店舗向け.xlsx`）の読み込み処理を `try-except` ブロックで囲み、ファイル欠落やシート名違いが発生した際、ユーザーにわかりやすい日本語エラーメッセージを表示して安全に終了するようにします。
