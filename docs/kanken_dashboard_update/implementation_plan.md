# Kankenダッシュボード(v6対応版) 実装計画

既存の静的な「Kanken.html」を元に、最新の「Kanken23510_2026年版_v6.xlsx」のデータを使って、店舗スタッフ向けのHTMLダッシュボードを自動生成・更新できる仕組みを構築します。

> [!NOTE]
> 現在の`Kanken.html`は固定の数値が直接書き込まれています。これを毎回手作業で書き換えるのは非効率なため、Pythonスクリプトを実行するだけで最新のExcelデータを読み込み、自動的に新しいHTMLファイルを出力する仕組みを提案します。店舗スタッフには、出力された新しいHTMLファイルを共有するだけで済みます。

## User Review Required

> [!IMPORTANT]
> 以下のロジックについての明確なルールをお教えください。ルールが確定次第、正確なスクリプトを実装します。

1. **「1〜5月 販売数昨対比 135%」の数値について**
   この「135%」という数値は、プログラム実行時に手動で毎回入力（またはスクリプト内の設定値として固定）する運用でよろしいでしょうか？それとも、Excelファイル内のどこかのシートや、別の実績CSVから自動計算する必要がありますか？

2. **カラーの分類ルール（しきい値）について**
   ダッシュボード上ではカラーが「優先提案」「コントロール大」「コントロール小」に分かれています。v6のデータでは、どのような条件で分類すればよろしいでしょうか？
   （例案：過不足が0以上なら「優先」、-500個以下なら「コントロール大」、それ以外は「コントロール小」にする、など明確なルールがあれば教えてください）

## Proposed Changes

### Dashboard Generator

#### [NEW] `update_kanken_dashboard.py`
Excelファイル（Kanken23510_2026年版_v6.xlsx）を読み込み、データを抽出して新しいダッシュボードHTML（例：`Kanken_Dashboard_latest.html`）を自動生成するPythonスクリプトです。

- **データ抽出**: 「全体サマリー」シートから、総供給、需要予測、過不足の合計値を取得します。
- **グラフ用データ**: 需要予測数が多い上位7カラーを自動抽出し、グラフ用のデータセットを構成します。
- **カラー分類**: ヒアリング結果に基づいた条件で、各カラーを3つのカテゴリーに自動分類します。
- **HTML出力**: Alpine.jsを用いた動的なHTMLテンプレート内にデータを埋め込み、1つのファイルとして出力します。

#### [MODIFY] `kanken.html` (テンプレート化)
現在の `kanken.html` をベースにしつつ、Pythonスクリプトがデータを注入しやすいように、内部のデータ定義部分を整理（テンプレート化）します。

## Verification Plan

### Manual Verification
1. スクリプト実行後、新たに生成された `Kanken_Dashboard_latest.html` をブラウザで開きます。
2. Excel「全体サマリー」シートの合計値（総供給、需要予測、全体過不足）と、画面上部のKPIが一致しているか確認します。
3. グラフが正常に描画され、上位7カラーが表示されているか確認します。
4. 各カラーが指定したルール通りに「優先」「コントロール大」「小」に正しく分類されているか確認します。
